Project Overview

This project demonstrates Database Testing using SQL for an E-Commerce application.
The main objective is to validate database integrity, verify business rules, detect data inconsistencies, and perform QA-focused SQL testing using realistic datasets.
The project is designed as a portfolio project for Software Quality Assurance (SQA) Engineers.

Objectives

  - Design an E-Commerce Database
  - Insert Sample Data
  - Perform SQL Testing
  - Validate Data Integrity
  - Detect Database Bugs
  - Practice SQL Queries
  - Demonstrate QA Database Testing Skills
